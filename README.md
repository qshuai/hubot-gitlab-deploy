# hubot-gitlab-deploy

gitlab deploy

See [`src/gitlab-deploy.coffee`](src/gitlab-deploy.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-gitlab-deploy --save`

Then add **hubot-gitlab-deploy** to your `external-scripts.json`:

```json
[
  "hubot-gitlab-deploy"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
